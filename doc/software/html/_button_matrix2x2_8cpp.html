Example program demonstrating the usage of 2x2 button matrix.

Author

Date

This is an example of an application with 2x2 button matrix. During the code, one pin is configured to input with pullup resistor One pin is configured as output sink to form the ground. the other two pins are set to output high to avoid interference.

In the example, the matrix was connected as follows: L1 -> PD2 L2 -> PD3 R1 -> PD4 R2 -> PD5
